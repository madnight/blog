Semigroup
An Associative Magma

A semigroup is an algebraic structure ( S , ⊗ ) (S, \otimes) (S,⊗) in which S S S is a non-empty set and ⊗ : S × S → S \otimes : S \times S \rightarrow S ⊗:S×S→S is a binary associative operation on S S S, such that the equation ( a ⊗ b ) ⊗ c = a ⊗ ( b ⊗ c ) (a \otimes b) \otimes c = a \otimes (b \otimes c) (a⊗b)⊗c=a⊗(b⊗c) holds for all a , b , c ∈ S a, b, c \in \mathcal{S} a,b,c∈S. In category theory, a semigroup is a monoid where there might not be an identity element. More formally, a semigroup is a semicategory (a category without the requirement for identiy morphism) C \mathcal{C} C with just one object S S S and the following conditions:

  • The set of morphisms (hom-set) is closed under composition: For every pair of morphisms f , g f, g f,g in H o m C ( S , S ) Hom_\mathcal{C}(S,S) HomC​(S,S), their composition f ∘ g f \circ g f∘g also belongs to H o m C ( S , S ) Hom_{C}(S,S) HomC​(S,S)

  • The composition operation is associative: For any three morphisms f , g , h f, g, h f,g,h in H o m C ( S , S ) Hom_{C}(S, S) HomC​(S,S), we have ( f ∘ g ) ∘ h = f ∘ ( g ∘ h ) (f \circ g) \circ h = f \circ (g \circ h) (f∘g)∘h=f∘(g∘h).

Example

A type qualifies as a Semigroup if it offers an associative function (<>), allowing the merging of any two type values into a single one.

All of the above is already implemented in the standard Haskell library, so you can also simply open an interactive Haskell interpreter (ghci) and test the following examples.

Some more examples are:

  • The naturals numbers without zero ( N , + ) ({\mathbb {N}},+) (N,+) under addition. This forms a semigroup because addition is associative.

  • The natural numbers ( N 0 , × ) ({\mathbb {N}}_{0}, \times ) (N0​,×) under multiplication. This forms a semigroup because multiplication is associative, but is also a monoid ( N 0 , × , 1 ) \left({\mathbb {N}}_{0},\times,1\right) (N0​,×,1) since 1 serves as the identity, as any number multiplied by 1 remains the same.

  • Non-empty strings ( String , + + ) (\texttt{String},++) (String,++) under concatenation. This forms a semigroup because string concatenation is associative.
